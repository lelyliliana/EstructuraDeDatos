# Estructura de Datos

En este repositorio se irán agregando los programas que servirán como ejemplo durante el desarrollo del curso **Estructura de Datos**.

Estos ejemplos tienen como propósito apoyar la comprensión de los temas vistos en clase, a través de la implementación práctica en diferentes lenguajes de programación.

---

## Contenido

- Programas de ejemplo por unidad o tema.
- Algoritmos básicos y estructuras fundamentales.
- Implementaciones paso a paso desarrolladas en clase.

---

## Propósito educativo

Este repositorio tiene fines académicos dentro del curso **Estructura de Datos**.
